Portable_XH

Portable_XH ist eine portable CMSimple_XH Distribution, die zur Ausführung auf einem lokalen Computer gedacht ist, ohne die Notwendigkeit einen Webserver mit PHP zur Verfügung zu haben. Der Zugriff von anderen Computern ist nicht möglich, so dass sie Ihre Firewall , wenn einkommende Anfragen von Ihrer Firewall blockiert werden.

Portable_XH ist die perfekte Lösung eine Demo von CMSimple_XH zur Hand zu haben, oder um eine Website portabel vorzubereiten, ohne diese gleich auf einen Webspace hoch laden zu müssen.
